# trapcitylabs-assets

Public-hosted brand assets for TrapCity Labs, used where a service needs a fetchable image URL (e.g. the Cloudflare Access login-page logo).

- `tcl-monogram-512.png` — TCL monogram, 512×512.

Source of truth for brand files is the private `trapcitylabs-llm` repo; this repo only mirrors what must be public.
